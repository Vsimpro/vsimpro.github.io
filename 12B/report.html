12 – Create a Website:
Sivun tyyli:
Päätin tehdä omat sivuni ajamaan kotisivujen roolia. Sivuilla esittelen nopeasti miten ne pyörivät sekä nopean kuvauksen IT:taidoistani.
Rakenne ja asettelu:
Sivujen rakenne on tehty hyvin yksinkertaiseksi. Landing page- nojaa täysin taustakuvaan, joka tehtiin varta vasten tätä käyttöä varten. Muu media on kasattu joko ennalta tehdyistä asseteista tai muiden valmiiksi rakentamista elementeistä (esim. SoMe badget.)
Validointi:
CSS ei antanut valitoidessa erroreita, mutta html ei suostunut validoimaan kuvien skaalausta prosenttien mukaan.
Itsearvio:
Antaisin itselleni 8 pistettä sivuista ja sisällöstä 2, siis kokonaispisteinä 10/12. Sivut ovat toimivat ja tyyli siisti sekä yhtenäinen.
