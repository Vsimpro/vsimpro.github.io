Did you know?

That this site is self-hosted!

Originally, these pages were made to be my final assignment on a html course.

But the thing is, depending on whether you're viewing this site on github.io or vsim.xyz, the .html files could be a bit different, let me explain.

Python has a module called Flask , which is used to create web applications. As you can see from the image above, it's relatively simple to get your project started with it. With mere 7 lines of code you can get your "Hello World!" out there.

However, Flask handles file locations and variables a bit differently. You're going to have to assign a folder for templates, and set it as the static folder for the project. Links will have to be in format,
{{ url_for("static", filename="") }}, with "filename" including the location for the file in the folder structure.

The code with all of it's beauty is tormenting running on a Raspberry Pi 4. These small things are amazing, I love them so much I bought 4 of them!
